Портал данных - для торговли, кооперации, инвестиций

Подключиться

Сервис ПорталДанных.РФ на уровне метаданных автоматически соединяет информацию о потребностях покупателей с предложениями поставщиков.

Еще никогда поиск клиентов и поставщиков не был таким простым и быстрым.

API интеграция в полном объеме

Вы можете подключить сервис к любым своим системам: веб-сайту, мобильному
приложению, боту, CRM, SRM, ERP, 1С продуктам и т.д.

Адреса

API работы с почтовыми адресами и геокоординатами.

Разбор адреса из строки («стандартизация»)

Разбивает адрес из строки по отдельным полям (регион, город, улица, дом, квартира) согласно КЛАДР / ФИАС (ГАР). Определяет почтовый индекс, часовой пояс...

Подробнее

Автодополнение при вводе («подсказки»)

Помогает человеку быстро ввести корректный адрес на веб-форме или в приложении.

Подробнее

Геокодирование

Определяет координаты по адресу из строки. Заодно возвращает почтовый индекс и вообще все данные по адресу.

Подробнее

Обратное геокодирование

Возвращает всю информацию об адресе по координатам. Работает для домов, улиц и городов.

Компания по ИНН
